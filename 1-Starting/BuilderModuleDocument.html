Builder Module

Welcome! Builder Module is a module that will help roblox developers to create building animations for models. Here, i'll teach you how to get started with Builder Module.

Note: This documentation is still in progress. If you find any issues, please file an issue on the Module GitHub page issue tracker!

Installation

All you need to get the module from this link: Roblox Marketplace Builder Module After getting the module, you need to put module inside ReplicatedStorage.

Starting Up

To create a building animation we need to firts define module in our script(it can work for client and server scripts).

local BuilderModule = require(path.to.your.module) --You need to create path to module like: local BuilderModule = require(game:GetService("ReplicatedStorage")["iyiinsan5555's_CustomBuilderModule"])

After that all you need to get ready your model in script.

Example script:

local BuilderModule :Module = require(game("ReplicatedStorage")["iyiinsan5555's_CustomBuilderModule"]) local ExampleModel :Model = workspace:WaitForChild("ExampleModel") BuilderModule.MakeModelReadyForBuild(ExampleModel)

Alright! After making model ready for build. You can create build animation whenever you want by calling this function:

BuilderModule.Build(BuildingMode :string, Model :Model, WaitTimeBetweenEachPart :number, CustomTweenInfo :TweenInfo)

Building Modes

You can pick one of the building mode by typing

BuilderModule.BuildingModes["Whatever You Want"]

The building mode is gonna change building animation.

List of building modes:

ClassicFadeOut

ComesFromRandomPosition

ComesFromRandomPositionWithFade

GrowthAnimation

GrowthAnimationWithFade

HorizontalAnimation

VerticalAnimation

ComesFromCenter

ComesFromCenterWithFade
