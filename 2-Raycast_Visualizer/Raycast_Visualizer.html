Raycast Visualizer

made by iyiinsan5555

What does this module?-What is the benefits of this module?

It visualizes raycast or spherecast with advanced settings.

It's good to show begginners how raycast and shperecast works.

It's also very good at visualizing raycast and spherecast.

Note: It doesn't support ray and blockcast

Installation

You need to get module from this line: iyiinsan5555_Raycast_Visualizer

Put it wherever you want. I suggest you to put it into Replicated Storage. So you can use it in server scripts and client scripts.

Setup

First of all you need to require module to use it. Like that:

local ReplicatedStorage = game:GetService("ReplicatedStorage") local Raycast_Visualizer = require(ReplicatedStorage:WaitForChild("iyiinsan5555_Raycast_Visualizer"))

Usage

To visualize a raycast we need:

Origin(Start Position), in Vector3. We also need Direction(Which direction raycast go and how long) in Vector3.

Let's get Vector3.new(0,5,0) as origin(start) and Vector3.new(100,0,0) as a direction.Code example:

local ReplicatedStorage = game:GetService("ReplicatedStorage") local Raycast_Visualizer = require(ReplicatedStorage:WaitForChild("iyiinsan5555_Raycast_Visualizer")) local Origin = Vector3.new(0,5,0) local Direction = Vector3.new(100,0,0)

After (belirlemek) origin and direction we can call function. Here is an example of calling function:

local ReplicatedStorage = game:GetService("ReplicatedStorage") local Raycast_Visualizer = require(ReplicatedStorage:WaitForChild("iyiinsan5555_Raycast_Visualizer")) local Origin = Vector3.new(0,5,0) local Direction = Vector3.new(100,0,0) Raycast_Visualizer:VisualizeRaycast(Origin, Direction)

As you can see we added this line to call function:

Raycast_Visualizer:VisualizeRaycast(Origin, Direction)

If you run the game you can see a beam that reperstand[temsil etmek] our raycast.

Info about how to use settings

If you did everything right you were able to see a beam (our raycast visualizer).

Let's learn how to customize our beam!

To customize beam we need to go to our module settings like that:

local ReplicatedStorage = game:GetService("ReplicatedStorage") local Raycast_Visualizer = require(ReplicatedStorage:WaitForChild("iyiinsan5555_Raycast_Visualizer")) Raycast_Visualizer.Settings

To change color of beam you need to change:

Raycast_Visualizer.Settings.BeamColor = Color3.fromRGB(255,0,0) --Feel free to change color

There is all settings that you can change:

Raycast_Visualizer.Settings.BeamColor = Color3.fromRGB(255,0,0) --To change beam color Raycast_Visualizer.Settings.BeamWidth = 0.2 --To change beam width Raycast_Visualizer.Settings.InstancesTransparency = 0.5 --Manages instance's (parts that module create) transparency Raycast_Visualizer.Settings.InstancesColor = Color3.fromRGB(100,100,100) --Manages instance's color Raycast_Visualizer.Settings.InstancesFolderName = "iyiinsan5555's_RaycastVisualizer" --Module creates a folder to hold instances. You can change it's name. Raycast_Visualizer.Settings.VisualizationLifeTime = 10 -- It manages the time that how long visualization lasts. Raycast_Visualizer.Settings.HitPartEffectTransparency = 0.75 --Module creates a part to show what your raycast hit. It changes part's transparency. Raycast_Visualizer.Settings.CreateBillboard = true --If you set it to true, module creates billboards to give info about raycast. Raycast_Visualizer.Settings.BillboardMaxDistance = 30 --The distance that you can see billboard.

Shperecast

Only difference between raycast and spherecast is radius.

Radius is the number that (belirleme) how large sphere should be.

Let's see how to use VisualizeSpherecast.

local ReplicatedStorage = game:GetService("ReplicatedStorage") local Raycast_Visualizer = require(ReplicatedStorage:WaitForChild("iyiinsan5555_Raycast_Visualizer")) local Origin = Vector3.new(0,5,0) local Direction = Vector3.new(100,0,0) local Radius = 2 Raycast_Visualizer:VisualizeSpherecast(Origin, Direction, Radius)
